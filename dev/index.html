Overview

InfrastructureSystems.jl is a Julia package that utilities for the packages in NREL's SIIP Initiative. This package is meant for package development. It is used primarily by PowerSystems.jl and PowerSimulations.jl but is written to be extensible for other kinds of applications as well within.

This document describes how to integrate it with other packages.
